# PendulumSim 用户操作手册

## 进入方式

- 首页或项目页进入「混沌摆模拟」。

## 基本操作

- 暂停/继续：右侧控制面板的「暂停 / 继续」按钮。
- 重置：恢复到当前参数对应的初始角度与静止状态。
- 拖拽：在模拟区域直接拖拽摆锤（支持鼠标/触控），松手后继续受力运动。

## 参数说明

- 模式：双摆（2 个摆锤）/ 三摆（3 个摆锤）。
- 重力 g：越大摆动越快。
- 阻尼 damping：模拟空气阻力，越大能量衰减越快（趋于稳定）。
- 摆长 L1/L2/L3：决定几何尺度与周期。
- 质量 m1/m2/m3：影响惯性与约束分配。
- 初始角度 θ1/θ2/θ3：单位为度，修改后会立即重置到新的初始姿态。
- 轨迹：开关与长度设置（过长可能影响低端设备帧率）。
- 能量曲线：显示动能、势能与总能（归一化），用于观察数值误差与能量漂移。

## 误差提示

- 左上角能量面板中的 “能量漂移” 表示当前总能相对初始总能的偏差（越接近 0% 越好）。
